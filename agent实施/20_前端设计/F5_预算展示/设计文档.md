# F5 预算展示设计

> 状态：✅ 已完成  
> 对应后端：P11 成本与预算控制  
> 完成日期：2026-05-01

---

## 1. 背景

后端 BudgetController 在 `done.metadata.budget_usage` 中返回任务的 Token 和工具调用消耗。前端需要展示这些数据，让用户了解任务成本。

## 2. 目标

- [ ] 任务完成后展示预算消耗摘要。
- [ ] 展示 Token 用量（input/output）。
- [ ] 展示各工具调用次数和预估成本。
- [ ] 预算警告提示（接近上限时高亮）。

## 3. 涉及文件

| 文件 | 类型 | 说明 |
|---|---|---|
| `components/BudgetSummary.vue` | 新增 | 预算摘要面板 |
| `components/TokenUsage.vue` | 新增 | Token 用量条 |

## 4. 数据来源

```json
{
  "metadata": {
    "budget_usage": {
      "input_tokens": 3200,
      "output_tokens": 1800,
      "tool_calls": 5,
      "calls_per_tool": {
        "generate_image": 2,
        "image_to_video": 1,
        "text_to_speech": 1,
        "merge_media": 1
      },
      "estimated_cost": 0.035
    }
  }
}
```

## 5. UI 布局

```text
┌─── 本次任务消耗 ───┐
│ Token: 3.2k / 1.8k │
│ 工具调用: 5 次      │
│ 预估成本: ¥0.035   │
│                    │
│ generate_image × 2 │
│ image_to_video × 1 │
│ text_to_speech × 1 │
│ merge_media    × 1 │
└────────────────────┘
```
