> 状态：✅ 已完成

> 完成日期：2026-05-01

---

## 1. 背景

后端已实现 `GET /tools/health` 和 `GET /tools/stats` API，前端需要工具管理面板展示工具能力元数据、健康状态和调用统计。

## 2. 目标

- [ ] 工具列表页：展示所有注册工具的元信息。
- [ ] 健康状态：显示每个工具的可用性（available/degraded/unavailable）。
- [ ] 调用统计：展示工具调用次数、成功率、平均耗时。
- [ ] Fallback 关系：展示工具的备选关系链。

## 3. 涉及文件

| 文件 | 类型 | 说明 |
|---|---|---|
| `views/comic-agent/ToolManagement.vue` | 新增 | 工具管理页 |
| `components/ToolCard.vue` | 新增 | 单个工具信息卡片 |
| `components/ToolStats.vue` | 新增 | 调用统计图表 |

## 4. API 调用

```typescript
// 工具健康
const health = await api.get('/comic-agent/tools/health')
```
