## 5. 拆分原则（实际执行）

- **单一职责**：每个组件/composable 只做一件事。
- **Props Down, Events Up**：子组件通过 props 接收数据，emit 事件通知父组件。
- **单例 Composable**：`useConfig` 使用模块级 ref，多组件调用共享同一状态。
- **渐进式拆分**：共 6 轮，每轮拆完 `vite build` 验证 + git commit。
- **样式分发**：scoped style 按组件归属拆分为 SCSS partial 文件。
