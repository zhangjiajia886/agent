# P11 成本与预算控制设计

> 所属阶段：P11  
> 状态：待实施  
> 目标：限制 Agent 长任务中的 token、工具调用、GPU/API、视频生成和存储成本，防止自动执行失控。

---

## 1. 背景

漫剧 Agent 会调用 LLM、ComfyUI、即梦、TTS、媒体合成等能力。复杂任务或失败重试可能造成资源消耗失控。

## 2. 目标

- [ ] 定义任务预算 `TaskBudget`。
- [ ] 定义预算消耗记录 `BudgetUsage`。
- [ ] LLM 调用前检查 token 预算。
- [ ] 工具调用前检查工具预算。
- [ ] 视频/图片生成前检查高成本预算。
- [ ] 预算不足时 ask_user/block。
- [ ] 记录每个任务的估算成本。

## 3. 非目标

- 首期不做真实支付扣费。
- 首期不做复杂套餐系统。
- 首期成本可估算，不要求精确财务计费。

## 5. TaskBudget

```python
@dataclass
class TaskBudget:
    max_iterations: int
    max_tool_calls: int
    max_calls_per_tool: dict[str, int]
    max_input_tokens: int
    max_output_tokens: int
    max_image_generations: int
    max_video_generations: int
    max_audio_generations: int
    max_duration_seconds: int
    max_estimated_cost: float
```

## 6. BudgetUsage

```python
@dataclass
class BudgetUsage:
    iterations: int = 0
    tool_calls: int = 0
    calls_per_tool: dict[str, int] = field(default_factory=dict)
    input_tokens: int = 0
    output_tokens: int = 0
    image_generations: int = 0
    video_generations: int = 0
    audio_generations: int = 0
    duration_seconds: int = 0
    estimated_cost: float = 0.0
```

## 7. 预算检查流程

```text
准备调用工具/LLM
  ↓
BudgetController.check(...)
  ├── allow → 执行
  ├── warn → 执行但记录 warning
  ├── ask_user → 请求用户确认继续消耗
  └── block → 阻止执行
```

## 8. 成本估算

工具成本等级：

| 工具类型 | 成本等级 | 示例 |
|---|---|---|
| read/list/grep | low | 本地只读 |
| text_to_speech | medium | TTS |
| generate_image | medium/high | ComfyUI/即梦图片 |
| image_to_video | high | 视频生成 |
| merge_media | medium | ffmpeg 合成 |
| LLM | variable | token 计费 |
