> 所属阶段：P0.5

## 1. 背景

P0 已实现内存态 `RuntimeTask / RuntimeStep`、标准 `ToolResult`、`task_created / step_update / artifact_created / done.final_report` 事件和前端消费逻辑。

但这些能力还没有经过端到端验证。

## 2. 目标

- [ ] 验证后端新增模型不会影响启动。
- [ ] 验证 WebSocket 普通聊天仍然可用。
- [ ] 验证工具任务能正确发出 P0 新事件。
- [ ] 验证前端工作台能展示后端 task/step/artifact/final_report。
- [ ] 验证工具失败不会误判 completed。

## 3. 非目标

- 不实现 DB 持久化。
- 不实现 Scheduler。
- 不实现 Replanner。
- 不重构 `agent_runner.py`。
