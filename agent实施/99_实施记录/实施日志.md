### P7 Sandbox 与安全隔离最小落地

- 新增 `backend/app/core/comic_chat_agent/sandbox.py`：
  - `RiskLevel`（L0-L4）工具风险等级枚举。
  - `TOOL_RISK_LEVELS` 全量工具 → 风险等级映射。
  - `SandboxPolicy` 可配置策略（路径 / 命令 / 网络 denylist）。
  - `SandboxChecker` 前置检查器：
    - `check_read_path` —— 禁止读取 .env / .ssh / .git 等。
    - `check_write_path` —— 仅允许 agent_outputs / /tmp/ttsapp_agent_tasks。
    - `check_command` —— 拦截 rm -rf / fork bomb / curl|sh 等。
    - `check_python_code` —— 拦截 os.environ / subprocess 等。
    - `check_url` —— 拦截 metadata IP / localhost 等。
    - `check_tool` —— 统一入口，按工具类型分派检查。
- 修改 `backend/app/core/comic_chat_agent/tool_executor.py`：
  - 在 `execute_tool()` 入口增加 `_sandbox.check_tool()` 前置检查。
  - 被拦截时返回 `status=blocked` + `error_code=SANDBOX_*_BLOCKED` + `risk_level`。
