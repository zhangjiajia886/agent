# P6 恢复与观测测试用例

## 用例 1：事件回放

- **操作**：完成一次任务后刷新页面
- **期望**：
  - [ ] 可查询 task events
  - [ ] 前端能重建 steps/artifacts/logs

## 用例 2：WebSocket 断线

- **操作**：任务执行中断开连接
- **期望**：
  - [ ] 重新连接后能看到当前任务状态
  - [ ] 已完成产物不丢失

## 用例 3：任务取消

- **操作**：调用 cancel API
- **期望**：
  - [ ] task.status=canceled
  - [ ] running step canceled 或 blocked

## 用例 4：工具调用追踪

- **检查**：
  - [ ] ToolInvocation 有 duration
  - [ ] 有 input/output/status/error
  - [ ] 前端时间线可展示
