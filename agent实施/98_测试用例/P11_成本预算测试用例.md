# P11 成本预算测试用例

## 用例 1：工具调用次数上限

- **预算**：max_tool_calls=3
- **操作**：触发第 4 次工具调用
- **期望**：
  - [ ] 调用被阻止或 ask_user
  - [ ] 产生 budget_warning/budget_blocked 事件

## 用例 2：视频生成次数上限

- **预算**：max_video_generations=1
- **操作**：尝试生成第 2 个视频
- **期望**：
  - [ ] 阻止自动执行
  - [ ] 提示用户是否继续消耗预算

## 用例 3：Token 预算上限

- **预算**：max_input_tokens/max_output_tokens 较低
- **操作**：长对话任务
- **期望**：
  - [ ] 触发摘要或停止
  - [ ] 不继续无限调用 LLM

## 用例 4：高成本工具 warning

- **工具**：image_to_video
- **期望**：
  - [ ] 调用前检查成本等级
  - [ ] 超过阈值时发出 warning

## 用例 5：final_report 包含预算摘要

- **任务结束后检查**：
  - [ ] final_report.usage 存在
  - [ ] 包含 tool_calls/token/video_generations/estimated_cost
