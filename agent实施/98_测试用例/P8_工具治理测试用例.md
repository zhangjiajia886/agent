# P8 工具治理测试用例

## 用例 1：核心工具能力元数据完整

- **检查工具**：generate_image/image_to_video/text_to_speech/merge_media
- **期望**：
  - [ ] 有 risk_level
  - [ ] 有 input_artifact_types
  - [ ] 有 output_artifact_types
  - [ ] 有 required_inputs
  - [ ] 有 timeout_seconds

## 用例 2：Planner 使用工具元数据

- **输入**：生成图并转视频
- **期望**：
  - [ ] Planner 根据 image_to_video.input_artifact_types 生成 depends_on
  - [ ] Planner 根据 generate_image.output_artifact_types 连接上游图片

## 用例 3：禁用工具不被选中

- **模拟**：禁用 image_to_video
- **期望**：
  - [ ] Planner 不生成可执行 image_to_video step
  - [ ] 有 fallback 时使用 fallback
  - [ ] 无 fallback 时 blocked

## 用例 4：fallback 可读取

- **工具**：generate_image
- **期望**：
  - [ ] fallback_tools 包含 jimeng_generate_image
  - [ ] Replanner 可读取 fallback_tools

## 用例 5：健康检查

- **模拟**：ComfyUI 不可达
- **期望**：
  - [ ] 工具 health_status=unhealthy/degraded
  - [ ] Planner/Scheduler 避免优先选择不可用工具
