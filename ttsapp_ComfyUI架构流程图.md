        │                           ┌──────────────────────┐

## 五、根因分析：为什么会有这么多错误？

```
                    ┌──────────────────────────────────┐
                    │  ComfyUI 编辑器导出 API JSON 时   │
                    │  widget_values 按数组索引映射      │
                    │  到 inputs 字段名                  │
                    └────────────────┬─────────────────┘
                                     │
                    ┌────────────────▼─────────────────┐
                    │  远程服务器升级了自定义节点版本     │
                    │  节点新增/删除/重排了参数           │
                    └────────────────┬─────────────────┘
                                     │
              ┌──────────────────────▼──────────────────────┐
              │              值错位（张冠李戴）               │
              │                                              │
              │  ┌─────────┐  ┌─────────┐  ┌─────────┐     │
              │  │ 数值→枚举 │  │ bool→路径│  │ int→连接│     │
              │  │ 512→bbox │  │ False→VAE│  │ 0→IMAGE │     │
              │  │ detector │  │ loader   │  │         │     │
              │  └────┬────┘  └────┬────┘  └────┬────┘     │
              │       │            │            │           │
              │  COMBO 验证   执行时 str    执行时 int      │
              │  400 拒绝     没有 .dim()   没有 .get()     │
              └──────────────────────────────────────────────┘
```

### 修复原则

1. **查远程节点定义** (`/object_info`) → 获取字段类型、范围、枚举选项
2. **COMBO 错误** → 改为节点定义中的默认值
3. **Optional 节点连接字段存了标量** → 删除该字段（让 ComfyUI 用 None）
4. **Required 标量字段值越界** → 改为节点定义中的默认值
5. **不可修的环境问题** → 记录为已知限制
