# 数据库和服务常用命令手册

## 🚀 Docker Compose 管理命令

### 启动所有服务
```bash
docker-compose up -d
```

### 停止所有服务
```bash
docker-compose down
```

### 查看服务状态
```bash
docker-compose ps
```

### 查看服务日志
```bash
# 查看所有服务日志
docker-compose logs -f

# 查看特定服务日志
docker-compose logs -f mysql
docker-compose logs -f redis
docker-compose logs -f elasticsearch
docker-compose logs -f milvus-standalone
```

### 重启服务
```bash
# 重启所有服务
docker-compose restart

# 重启特定服务
docker-compose restart mysql
docker-compose restart redis
```

## 📊 MySQL 常用命令

### 连接信息
- **Host**: localhost
- **Port**: 3306
- **Root密码**: root123456
- **数据库**: ttsapp
- **用户**: ttsapp
- **密码**: ttsapp123

### 连接 MySQL
```bash
# 使用 root 用户连接
docker exec -it ttsapp-mysql mysql -uroot -proot123456

# 使用 ttsapp 用户连接
docker exec -it ttsapp-mysql mysql -uttsapp -pttsapp123 ttsapp
```

### 常用 SQL 命令
```sql
-- 查看所有数据库
SHOW DATABASES;

-- 使用数据库
USE ttsapp;

-- 查看所有表
SHOW TABLES;

-- 查看表结构
DESC table_name;

-- 创建表示例
CREATE TABLE users (
    id INT PRIMARY KEY AUTO_INCREMENT,
    username VARCHAR(50) NOT NULL,
    email VARCHAR(100),
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 查询数据
SELECT * FROM users;

-- 插入数据
INSERT INTO users (username, email) VALUES ('test', 'test@example.com');

-- 更新数据
UPDATE users SET email = 'new@example.com' WHERE id = 1;

-- 删除数据
DELETE FROM users WHERE id = 1;
```

### 备份和恢复
```bash
# 备份数据库
docker exec ttsapp-mysql mysqldump -uroot -proot123456 ttsapp > backup.sql

# 恢复数据库
docker exec -i ttsapp-mysql mysql -uroot -proot123456 ttsapp < backup.sql
```

### 常用命令
```bash
# 设置键值
SET key "value"
SET user:1:name "张三"

# 获取值
GET key
GET user:1:name

# 设置带过期时间的键（秒）
SETEX session:abc123 3600 "user_data"

# 检查键是否存在
EXISTS key

# 删除键
DEL key

# 查看所有键（生产环境慎用）
KEYS *

# 查看键的过期时间
TTL key

# 列表操作
LPUSH mylist "item1"
RPUSH mylist "item2"
LRANGE mylist 0 -1

# 哈希操作
HSET user:1 name "张三" age 25
HGET user:1 name
HGETALL user:1

# 集合操作
SADD myset "member1" "member2"
SMEMBERS myset

# 有序集合
ZADD leaderboard 100 "player1" 200 "player2"
ZRANGE leaderboard 0 -1 WITHSCORES

# 查看数据库信息
INFO
INFO memory
INFO stats

# 清空当前数据库
FLUSHDB

# 清空所有数据库（慎用）
FLUSHALL
```

## 🔍 Elasticsearch 常用命令

### 连接信息
- **Host**: localhost
- **Port**: 9200
- **无需认证**

### 健康检查
```bash
# 集群健康状态
curl http://localhost:9200/_cluster/health?pretty

# 节点信息
curl http://localhost:9200/_nodes?pretty

# 查看所有索引
curl http://localhost:9200/_cat/indices?v
```

### 索引操作
```bash
# 创建索引
curl -X PUT "http://localhost:9200/my_index" -H 'Content-Type: application/json' -d'
{
  "settings": {
    "number_of_shards": 1,
    "number_of_replicas": 0
  }
}'

# 删除索引
curl -X DELETE "http://localhost:9200/my_index"

# 查看索引映射
curl http://localhost:9200/my_index/_mapping?pretty
```

### 文档操作
```bash
# 添加文档
curl -X POST "http://localhost:9200/my_index/_doc/1" -H 'Content-Type: application/json' -d'
{
  "title": "测试文档",
  "content": "这是一个测试内容",
  "timestamp": "2024-01-01T00:00:00"
}'

# 获取文档
curl http://localhost:9200/my_index/_doc/1?pretty

# 更新文档
curl -X POST "http://localhost:9200/my_index/_update/1" -H 'Content-Type: application/json' -d'
{
  "doc": {
    "title": "更新后的标题"
  }
}'

# 删除文档
curl -X DELETE "http://localhost:9200/my_index/_doc/1"
```

### 搜索操作
```bash
# 搜索所有文档
curl http://localhost:9200/my_index/_search?pretty

# 条件搜索
curl -X GET "http://localhost:9200/my_index/_search" -H 'Content-Type: application/json' -d'
{
  "query": {
    "match": {
      "title": "测试"
    }
  }
}'

# 批量操作
curl -X POST "http://localhost:9200/_bulk" -H 'Content-Type: application/json' -d'
{"index":{"_index":"my_index","_id":"1"}}
{"title":"文档1","content":"内容1"}
{"index":{"_index":"my_index","_id":"2"}}
{"title":"文档2","content":"内容2"}
'
```

## 🎯 Milvus 常用命令

### 连接信息
- **Host**: localhost
- **Port**: 19530
- **管理界面**: http://localhost:9001 (MinIO)

### Python SDK 使用示例
```python
# 安装 SDK
pip install pymilvus

# 连接 Milvus
from pymilvus import connections, Collection, FieldSchema, CollectionSchema, DataType

connections.connect(host="localhost", port="19530")

# 创建集合
fields = [
    FieldSchema(name="id", dtype=DataType.INT64, is_primary=True, auto_id=True),
    FieldSchema(name="embedding", dtype=DataType.FLOAT_VECTOR, dim=128)
]
schema = CollectionSchema(fields=fields, description="test collection")
collection = Collection(name="test_collection", schema=schema)

# 插入向量
import random
vectors = [[random.random() for _ in range(128)] for _ in range(100)]
collection.insert([vectors])

# 创建索引
index_params = {
    "metric_type": "L2",
    "index_type": "IVF_FLAT",
    "params": {"nlist": 128}
}
collection.create_index(field_name="embedding", index_params=index_params)

# 加载集合
collection.load()

# 搜索
search_params = {"metric_type": "L2", "params": {"nprobe": 10}}
results = collection.search(
    data=[vectors[0]], 
    anns_field="embedding", 
    param=search_params, 
    limit=10
)

# 查看集合列表
from pymilvus import utility
print(utility.list_collections())

# 删除集合
collection.drop()
```

## ⚠️ 注意事项

1. **生产环境**: 请修改所有默认密码
2. **资源限制**: Elasticsearch 和 Milvus 较占内存，建议至少 8GB RAM
3. **数据持久化**: 所有数据存储在 Docker 卷中，删除卷会丢失数据
4. **网络访问**: 默认只监听 localhost，如需外部访问请修改配置
5. **备份策略**: 定期备份重要数据
