## 七、常见问题排查

| 现象 | 原因 | 解决 |
|---|---|---|
| Singer 提交后一直「排队中」 | SSH 隧道断开 | `ssh -CNg -L 7862:127.0.0.1:6006 autodl-singer` |
| Podcast 提交失败 | AutoDL 实例休眠 | 登录 AutoDL 控制台 → 启动 RTX 5090 实例 |
| FlashHead 生成失败 | FlashHead 服务未运行 | `ssh autodl 'bash ~/start_all.sh'` |
| 登录页面空白 | 前端未启动 | `./start.sh restart` |
| 接口 401 错误 | Token 过期 | 退出重新登录 |

## 八、服务重启速查

```bash
# 重启本地 ttsapp
./start.sh restart

# 重启 RTX 5090 实例上的服务（FlashHead + Podcast）

# 重启 Singer（V100 实例）
ssh autodl-singer 'bash ~/start_singer.sh'

# 重建 Singer SSH 隧道
ssh -CNg -L 7862:127.0.0.1:6006 autodl-singer &
```
