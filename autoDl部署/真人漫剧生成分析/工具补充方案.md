# 漫剧 Agent 通用工具补充方案

## 1. 问题分析

### 现状：backend 工具只有 8 个漫剧专用工具

| # | 工具名 | 类型 | 执行器 |
|---|--------|------|--------|
| 1 | generate_image | 文生图 | comfyui |
| 2 | generate_image_with_face | 人脸保持 | comfyui |
| 3 | edit_image | 图像编辑 | comfyui |
| 4 | image_to_video | 图生视频 | comfyui |
| 5 | text_to_speech | 语音合成 | tts |
| 6 | upscale_image | 超分 | comfyui |
| 7 | merge_media | 媒体合成 | local |
| 8 | add_subtitle | 字幕叠加 | local |

**缺失能力**: 文件读写、Python 执行、Shell 命令、网络搜索、网页抓取、目录浏览、文件搜索、HTTP 请求等全部缺失。

### 参照：backend-myagent2 拥有 40 个工具 + 100+ 别名

## 2. 补充工具清单

### P0 - 基础能力（必须有）

| # | 工具名 | 描述 | executor_type | 来源参考 |
|---|--------|------|---------------|----------|
| 1 | `bash` | Shell 命令执行 | local | builtin.py |
| 2 | `read_file` | 读取文件内容 | local | builtin.py |
| 3 | `write_file` | 写入文件 | local | builtin.py |
| 4 | `python_exec` | 执行 Python 代码 | local | builtin.py |
| 5 | `web_search` | DuckDuckGo 搜索 | local | builtin.py |
| 6 | `web_fetch` | 获取网页内容 | local | builtin.py |

### P1 - 文件操作增强

| # | 工具名 | 描述 | executor_type |
|---|--------|------|---------------|
| 7 | `edit_file` | str_replace 精确编辑 | local |
| 8 | `grep_search` | 正则搜索文件内容 | local |
| 9 | `find_files` | Glob 模式搜索文件 | local |
| 10 | `list_dir` | 列出目录内容 | local |
| 11 | `http_request` | HTTP 请求(GET/POST等) | local |

### P2 - 可选高级工具（暂不实施）

- `git` / `image_read` / `pdf_read` / `env_info` / `zip_files`
- `mysql_query` / `excel_read` / `draw_diagram` 等

## 3. 架构方案

### 保持现有 DB-based 架构，最小改动

```
SEED_TOOLS[] → DB tool_registry → build_tool_definitions() → LLM
                                                                  ↓
tool_executor.py ← execute_tool(name, params) ← agent_runner.py
```

改动点：
1. **SEED_TOOLS** 增加 11 个通用工具的 JSON Schema 定义
2. **tool_executor.py** 增加 11 个通用工具的执行器实现
3. **TOOL_ALIASES** 增加通用工具的别名映射（从 myagent2 移植）
4. **System Prompt** 更新，让 Agent 知道自己有通用能力

### 文件变更清单

| 文件 | 改动 |
|------|------|
| `backend/app/api/v1/comic_agent.py` | SEED_TOOLS 增加 11 项 |
| `backend/app/core/comic_chat_agent/tool_executor.py` | 增加通用工具执行函数 + 别名 |
| `backend/app/api/v1/comic_agent.py` | system prompt 更新 |

## 4. 别名映射（从 myagent2 移植关键别名）

```python
# Python 执行
"execute_code" → "python_exec"
"run_python" → "python_exec"

# Shell
"shell_exec" → "bash"
"run_shell" → "bash"

# 文件读写
"create_file" → "write_file"
"save_file" → "write_file"
"read_file_content" → "read_file"

# 网络搜索
"search_web" → "web_search"
"google_search" → "web_search"

# 文件搜索
"search_files" → "grep_search"
"grep" → "grep_search"
"find_file" → "find_files"

# HTTP
"make_request" → "http_request"
"curl" → "http_request"
```

## 5. 实施顺序

1. tool_executor.py 新增通用工具执行函数
2. SEED_TOOLS 新增工具 Schema
3. TOOL_ALIASES 补充别名
4. System Prompt 更新
5. 重启后端 → DB 自动同步新工具
