## 三、前端操作流程图

### 流程A：基础图像生成（Z-Image / 双截棍）

```
开始
  │
  ▼
打开 ComfyUI 浏览器
  │
  ▼
点击菜单 [Load] 按钮
  │  选择文件：
  │  Z-Image系列/xxx.json
  │  或 双截棍-RTX5090/xxx.json
  ▼
工作流加载到画布
  │
  ▼
找到"正向提示词"节点（蓝色文本框）
  │  双击 → 编辑 → 输入提示词
  │  例：a beautiful xianxia girl, flowing robes, masterpiece
  ▼
找到"负向提示词"节点（橙色文本框）
  │  双击 → 确认/修改负向提示词
  ▼
找到 KSampler 节点
  │  调整：steps=20, cfg=7.0, seed=随机
  ▼
点击右上角 [Queue Prompt]
  │  等待进度条完成（Z-Image约3~5秒，双截棍约20~30秒）
  ▼
预览区显示生成图像
  │
  ▼
右键图像 → [Save Image] 保存
  │
  ▼
结束（记录耗时和效果评分）
```

### 流程B：人脸保持漫画生成（InstantID）

```
开始
  │
  ▼
准备：清晰正面人脸照片（本地，PNG/JPG，≥512×512）
  │
  ▼
Load InstantID 工作流（SD图像系列 或 其他图像 目录）
  │
  ▼
找到"Load Image"节点（人脸输入）
  │  点击节点上传图标 → 选择人脸照片
  ▼
找到"InstantID Strength"控制节点
  │  初始值设为 0.8
  ▼
编辑正向提示词节点
  │  输入风格描述（不要写人脸特征，让InstantID控制脸）
  │  例：xianxia style, ancient chinese female warrior,
  │       flowing red dress, mountain background, masterpiece
  ▼
确认 Checkpoint 选择
  │  推荐：NetaYumev35 或 majicMIX_realistic_v7
  ▼
确认 LoRA 节点
  │  推荐：仙侠风格.safetensors，强度 0.7
  ▼
[Queue Prompt]，等待约 20~40秒
  │
  ▼
评估人脸相似度（1-5分）
  │
  ├── 相似度 < 3 → 提升 InstantID Strength 到 1.0，重试
  ├── 相似度 3~4 → 当前值可用
  └── 相似度 = 5 → 完美，记录参数
  │
  ▼
保存图像，记录参数组合
```
