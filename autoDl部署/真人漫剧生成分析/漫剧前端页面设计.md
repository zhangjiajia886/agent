# 漫剧工作台 · 前端页面设计方案

> 目标：**让中国用户一看就会用**，无学习成本，3步完成漫剧创作。  
> 设计原则：功能分层呈现，核心流程前置，高级功能按需展开。

---

## 一、总体实现计划（分三期）

```
第一期（已完成 ✅）
  ├── Tab1 漫剧创作：文字描述 → AI 自动分镜 → 逐格出图
  ├── Tab2 图像编辑：UI 骨架（待接后端）
  └── Tab3 动态视频：UI 骨架（待接后端）

第二期（近期开发）
  ├── Tab1 补强：InstantID 工作流，人脸上传真正生效
  ├── Tab2 上线：Qwen 自然语言编辑图像
  └── 历史记录页：查看所有创作历史，支持续用

第三期（远期规划）
  ├── Tab3 上线：Wan I2V 图片动态化
  ├── 配音联动：漫剧图 + TTS 配音 = 完整短片
  └── 风格库页：一键套用预置风格，无需手写描述
```

---

## 二、核心用户旅程（3步法）

```
步骤 1  描述故事（10秒）
         └── 输入一句话：「仙侠风格4格漫剧，少女初入仙山，遇见神兽」
             或点击风格标签自动填充示例

步骤 2  等待生成（1~3分钟）
         └── 实时显示：分镜规划 → 第1格生成中 → 第2格 → ...
             每格完成后立即展示，不用等全部完成

步骤 3  下载使用（5秒）
         └── 点击下载全部图片，或右键保存单张
             可选：继续动态化 / 图像编辑 / 配音
```

```

│  ┌────────────────────────────────┐        │  │                                           │
│  │                                │        │  │  （左侧点击编辑后显示）                   │
│  │   [图片预览]  × 删除            │        │  │                                           │
│  │                                │        │  │                                           │
│  └────────────────────────────────┘        │  │  指令示例（点击快速填充）：               │
│                                            │  │  → 将背景替换为云雾缭绕的仙山            │
│  💬 编辑指令（用中文描述你想怎么改）        │  │  → 把衣服改为红色汉服                   │

│  │ 保持人物不变                    │        │  │  → 生成该角色的侧面视图                 │
│  └────────────────────────────────┘        │  │                                           │
│  200字以内                                 │  └───────────────────────────────────────────┘
│                                            │
│  ┌──────────────────────────────────┐      │  （编辑完成后）
│  │  ✨ 开始编辑                      │      │  ┌───────────────────────────────────────────┐
```

- 编辑历史面板：展示每次编辑前后对比（可回退）
- 指令示例标签点击即可填充输入框

```
│  │  拖入漫剧图片，或               │        │  │  （生成中）                               │
│  │  从「漫剧创作」结果中选择        │        │  │  ████████░░░░░░░░░  30%                  │
│  │                                │        │  │  🎬 Wan I2V 生成中，约还需 3 分钟...      │
│  └────────────────────────────────┘        │  │                                           │
│                                            │  │  💡 提示：生成完成后可加入 TTS 配音        │
│  🎭 运动描述（可选）                        │  │                                           │
│  ┌────────────────────────────────┐        │  └───────────────────────────────────────────┘
│  │ gentle hair swaying,           │        │
│  │ soft breathing                 │        │  （完成后）
│  └────────────────────────────────┘        │  ┌───────────────────────────────────────────┐
│  不填则 AI 自动决定                         │  │  ▶ [视频播放器]                           │
│                                            │  │                                           │

│                                            │  │  [下载视频]  [加入配音 →]                 │
│  ┌──────────────────────────────────┐      │  └───────────────────────────────────────────┘
```

**交互细节**：
- 支持批量动态化（一次上传多张，依次排队生成）
- 完成后直接跳转 TTS 配音功能（打通已有能力）
- 耗时较长，提供关闭页面后台生成+完成通知功能

## 四、通俗化设计原则（中国用户体验）

### 4.1 文案规范

| 技术名词 | 展示给用户的说法 |
|---|---|
| ComfyUI workflow | 创作风格 / AI 引擎 |
| InstantID | 人脸保留 / 保持我的脸 |
| LoRA | 风格滤镜 |
| Wan I2V | 图片变视频 |
| Qwen-Image-Edit | AI 改图 |
| prompt | 创作描述 |
| negative prompt | （不展示给用户）|
| seed | （不展示，自动处理）|

### 4.2 关键引导文案

```
漫剧创作页：
  → 「描述你想要的故事，AI 帮你画出来」
  → 「仙侠、水墨、盲盒……10多种风格一键切换」
  → 「上传一张你的照片，生成专属漫画形象」

图像编辑页：
  → 「告诉 AI 你想改什么，它来动手」
  → 「换背景、换服装、换风格，一句话搞定」

动态视频页：
  → 「让你的漫剧动起来！」
  → 「一张图片，变成 2 秒动态视频」
```

### 4.3 错误提示规范

| 场景 | 展示文案 |
|---|---|
| ComfyUI 服务断开 | 「AI 绘图服务暂时休息中，请稍后再试 🔄」|
| 生成超时 | 「AI 正在努力绘制中，请耐心等待 ⏳」|
| 人脸图片质量差 | 「请上传清晰正面照，效果更好哦 📸」|
| 描述为空 | 「说说你的故事吧，AI 来帮你画！」|

## 八、UI 改进建议（让用户更直觉）

### 8.1 首次进入引导

用户第一次打开漫剧工作台时，显示3步引导弹窗：

```
第 1 步：描述你的故事 → 点击「✨ 生成漫剧」
第 2 步：等待 AI 生成（约 1~3 分钟，可关闭页面）
第 3 步：下载图片，或继续编辑 / 动态化
```

### 8.2 示例画廊

在占位区展示已生成的优质样例（仙侠 / 水墨 / 盲盒各 1 组），  
用户点击示例可直接复用该故事描述，降低使用门槛。

### 8.3 生成完成通知

- 浏览器通知（需授权）：「✅ 你的漫剧已生成完毕，点击查看」
- 页面 title 闪烁提示
